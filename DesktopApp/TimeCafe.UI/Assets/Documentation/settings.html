Настройки приложения.

Страница предназначена для управления основными параметрами работы приложения, персонализации интерфейса и доступа к справочной информации.

Персонализация интерфейса.

  • Выбирать тему оформления приложения: светлая или тёмная.
  • Применять выбранную тему ко всем элементам интерфейса.

Настройка рабочего времени.

  • Устанавливать время открытия и закрытия заведения.
  • Сохранять изменения рабочего времени для корректной работы всех функций учёта посещений.

Минимальное время для входа.

  • Задавать минимальное время пребывания клиента в заведении (в минутах).
  • Сохранять изменения для применения новых ограничений.

Информация о приложении.

  • Просматривать сведения о версии и лицензии приложения.
  • Переходить по ссылке для ознакомления с политикой конфиденциальности и условиями использования.

Доступ к справке.

  • Открывать справочную систему приложения для получения подробных инструкций по работе с каждым разделом.
  • Использовать справку для быстрого поиска ответов на часто возникающие вопросы.
