Управление посетителями.

Страница предназначена для отображения и управления списком активных посетителей заведения. Необходимо использовать доступные инструменты для поиска, фильтрации и контроля времени пребывания клиентов.

Основные функции страницы.

  • Отображать список всех посетителей, находящихся в заведении в текущий момент.
  • Выполнять поиск по имени или фамилии клиента с помощью текстового поля поиска.
  • Применять фильтрацию по тарифу с использованием выпадающего списка.
  • В режиме реального времени вычислять продолжительность пребывания каждого посетителя и стоимость услуги согласно выбранному тарифу.
  • Просматривать информацию о времени входа, тарифе, стоимости и контактных данных клиента.
  • Осуществлять завершение посещения (выход клиента) с подтверждением действия.
  • Отслеживать общее количество активных посетителей.

Поиск по клиентам.

Для поиска клиента требуется: ввести имя или фамилию в поле поиска. Список посетителей автоматически обновляется с учетом введённого значения.

Фильтрация по тарифу.

Для применения фильтрации по тарифу требуется: выбрать нужный тариф из выпадающего списка. При выборе значения «Все тарифы» отображаются все посетители независимо от тарифа.

Вычисление времени и стоимости.

Для каждого посетителя система автоматически рассчитывает продолжительность пребывания с момента входа и текущую стоимость услуги. Расчёт выполняется в реальном времени с учетом выбранного тарифного плана и типа тарификации (почасовая или поминутная).

Дополнительные возможности.

  • Просматривать контактные данные клиента и сведения о тарифе.
  • Использовать кнопку «Выход» для завершения посещения клиента. При нажатии требуется подтвердить действие во всплывающем окне.
  • Контролировать общее количество посетителей с помощью соответствующего индикатора в верхней части страницы.
